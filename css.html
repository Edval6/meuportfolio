CSS

Como vimos anteriormente, o HTML é a ferramenta que nos permite criar uma pagina web e inserir conteúdos nelas, tais como textos, imagens, áudios, vídeos, etc. No entanto o HTML, por si só, não nos permite alterar a cor dos textos, o tamanho, o posicionamento, entre varias outras propriedades e é aqui onde entra o css.

CSS – Cascading Style Sheet ou Folha de Estilo em Cascata é a ferramenta que nos permite formatar a nossa página web e elementos inseridos, tais como textos, imagens, vídeos, áudios, etc.

Através do Css podemos criar estilos nas nossas páginas web por forma a oferecer uma navegação confortável aos nossos usuários.

Portanto o cus. e uma ferramenta complementar ao HTML e indispensável na criação de websites, porque e ela que permite a organização e a beleza da nossa pagina web.
