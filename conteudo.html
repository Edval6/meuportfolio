Textos em Html e Posicionamento com Css

No video de hoje vou ensinar como criar textos com html e css vamos abrir o vs code e o navegador criar o ficheiro html e css ligar o html com o css No html e possivel criar textos utilizando a tag h1 titulo1 e a tag p paragrafo. Primeiramente vamos conhecer os niveis dos titulos eles vao desde o h1, h2, ate o h6

vou agora criar os titulos para apresentar no navegador conforme podemos ver, a medida em que os niveis aumentam de h1, h2, etc... o tamanho da sua fonte vai diminuindo. Isso acontece pk o h1 e o maior titulo e o h6 e o menor titulo. de seguida vamos escrever o conteudo do nosso titulo, para tal vamos ao nosso html e vamos utilizar a tag p

com essa tag nos podemos escrever os nossos paragrafos dos nossos documentos Tendo aprendido a criar os titulos e o paragrafo, vamos agora aprender a formata-los utiilzando o css, o nosso outro ficheiro que foi criado. Dentro do css, nos vamos selecionar o elemento que quereos formatar, neste caso e o h1. de seguida abrimos colchetes e dentro deles vamos definir as suas propriedades Comecaremos com a font-size, que indica o tamanho da nossa fonte de seguida a font-family que indica o tipo da nossa font, arial, etc...
